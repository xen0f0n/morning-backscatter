# The Morning Backscatter

**The Morning Backscatter** is a quick morning overpass of the remote sensing and geospatial world.

It is a lightweight companion to [Spectral Reflectance](https://www.spectralreflectance.space/): shorter, more visual, and easier to scan.

Each issue follows a simple four-card format:

- **Pulse**: one main EO/geospatial news item
- **Quicklook**: one scene, map, image, or visual worth opening
- **Coherence**: one useful read, dataset, tutorial, or technical resource
- **Double Bounce**: one funny-ish or odd item from the EO world


## Where to read it

Website:

https://morningbackscatter.space/


RSS:

https://morningbackscatter.space/rss.xml

## When to expect new issues

The plan is to publish **twice a week**, keeping each issue short enough to read as part of a morning routine.


## How to follow / subscribe

New issues are published through **GitHub Releases**.

This is the closest thing to subscribing for now. I'm deliberately avoiding a mailing list or another publishing platform so that the project stays lightweight, both for readers and for me to maintain.


To follow along:

1. Open this repository on GitHub.
2. Click the **Watch** button near the top-right of the repo page.
3. Choose **Custom**.
4. Select **Releases**.
5. Save your notification preferences.

After that, GitHub should notify you when a new Morning Backscatter issue is released.
